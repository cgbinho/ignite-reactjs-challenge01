## Install & Run

```sh
  # clone repository
  git clone https://github.com/cgbinho/ignite-reactjs-challenge02 project-name

  # enter web folder
  cd project-name

  # download dependencies
  yarn

  # running / dev:
  yarn dev

  # create build
  yarn build

  # run tests
  yarn test
```
